Microflow Studio is built for two-hand use: one on the mouse, one on the keyboard. The shortcuts below are the full list.

<Callout type="info" title="macOS vs. Windows/Linux">
`Cmd` on macOS = `Ctrl` on Windows and Linux. Where this page writes `Cmd/Ctrl`, use whichever one your platform expects.
</Callout>

## Canvas

| Shortcut | Action |
| --- | --- |
| `n` | Open the **Add node** panel |
| `Space` (held) + drag | Pan the canvas |
| `Cmd/Ctrl + scroll` | Zoom |
| `Cmd/Ctrl + 0` | Reset zoom to 100 % |
| `Cmd/Ctrl + 1` | Fit all nodes in view |

## Nodes

| Shortcut | Action |
| --- | --- |
| Click | Select |
| Shift + click | Add to selection |
| Drag | Move selected node(s) |
| `Delete` / `Backspace` | Remove selected node(s) and their edges |
| `Cmd/Ctrl + D` | Duplicate selected node(s) |
| `Cmd/Ctrl + A` | Select all |
| Double-click | Open settings panel |

## Edges

| Shortcut | Action |
| --- | --- |
| Click and drag handle → handle | Draw an edge |
| Click edge | Select |
| `Delete` / `Backspace` | Remove selected edge |

## File

| Shortcut | Action |
| --- | --- |
| `Cmd/Ctrl + N` | New flow |
| `Cmd/Ctrl + O` | Open a `.microflow` file |
| `Cmd/Ctrl + S` | Save (overwrite current file) |
| `Cmd/Ctrl + Shift + S` | Save as… |
| `Cmd/Ctrl + Z` | Undo |
| `Cmd/Ctrl + Shift + Z` | Redo |

## Search

| Shortcut | Action |
| --- | --- |
| `Cmd/Ctrl + K` | Quick-find a node by name |
| `Cmd/Ctrl + F` | Find a node *on the current canvas* |
