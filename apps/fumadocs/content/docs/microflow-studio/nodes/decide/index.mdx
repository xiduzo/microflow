---
title: Decide
description: Decide nodes add logic to your flow — comparisons, counting, delays, and gates.
---

Decide nodes control what happens and when. They do not read sensors or drive outputs directly — they route, count, delay, and apply logic to the data flowing through your project.

Use them to:

- Check whether a value is above a threshold
- Count how many times something happened
- Wait a moment before passing a signal through
- Combine conditions with AND, OR, and other logic

| Node | What it does |
| --- | --- |
| [Compare](/docs/microflow-studio/nodes/decide/compare) | Checks if a value meets a condition and routes the flow accordingly |
| [Counter](/docs/microflow-studio/nodes/decide/counter) | Counts events — increments, decrements, or resets |
| [Delay](/docs/microflow-studio/nodes/decide/delay) | Pauses a signal for a set amount of time before passing it on |
| [Gate](/docs/microflow-studio/nodes/decide/gate) | Applies Boolean logic — AND, OR, NOT, NAND, NOR, XOR, XNOR |
