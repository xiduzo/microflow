---
title: Express
description: Express nodes output to the physical world — lights, sounds, motors, and external services.
---

Express nodes are the endpoints of your flow — they make things happen. Whether that is turning on a light, playing a sound, moving a motor, or publishing a value to other tools over MQTT, Express nodes are where your flow meets the real world.

Express nodes can control both:

- **Digital** outputs — simply on or off, like switching an LED
- **Analog** outputs — variable levels, like setting LED brightness or servo position

| Node | What it does |
| --- | --- |
| [MQTT (Express)](/docs/microflow-studio/nodes/express/figma) | Publishes a value to an MQTT topic — used by the Figma Hardware Bridge and Penpot plugin |
| [LED](/docs/microflow-studio/nodes/express/led) | Controls an LED — single, RGB, or LED matrix |
| [Piezo](/docs/microflow-studio/nodes/express/piezo) | Plays tones and melodies through a buzzer |
| [Relay](/docs/microflow-studio/nodes/express/relay) | Switches high-voltage circuits on or off |
| [Servo](/docs/microflow-studio/nodes/express/servo) | Controls a servo motor's position or speed |
| [Vibration](/docs/microflow-studio/nodes/express/vibration) | Drives a vibration motor for haptic feedback |
