Nodes are the building blocks of a flow. Each node does one specific thing — reading a sensor, turning on a light, doing math, making a decision — and you connect them together to build something interactive.

You connect nodes using [edges](/docs/microflow-studio/edges) (the lines between them). Data and signals travel along these edges from one node to the next.

**Example:** A light sensor reads brightness → a Map node scales the value → an LED adjusts its brightness accordingly.

## Adding a node to the canvas

Right-click anywhere on the blank canvas to open the node menu. You will see all available nodes organised by category. Click any node to place it on the canvas.

Once a node is on the canvas, you can:

- **Move it** by clicking and dragging
- **Configure it** by clicking the settings icon in its top right corner
- **Delete it** by selecting it and pressing the Delete or Backspace key

## Anatomy of a node

Every node has the same basic structure.

<Callout type="info" title="Image placeholder">
  Diagram showing the anatomy of a node: input handles on the left (blue), output handles on the right (green), the node value display in the center, and the settings icon in the top right corner.
</Callout>

```
   ┌─────────────────────── ⚙ settings
   │
   ▼
┌──────────────────────────┐
●  │        Node name        │  ●
●  │      [ live value ]     │  ●
●  │                          │  ●
└──────────────────────────┘
▲                              ▲
│                              │
input handles                  output handles
(blue, on the left)            (green, on the right)
```

**Input handles** — the blue dots on the left side. Connect other nodes here to send data into this node.

**Output handles** — the green dots on the right side. Connect these to other nodes to pass data onward. Different handles carry different types of signals (e.g. "pressed" vs "released" for a button).

**Node value** — the display in the center of the node. Shows the current value or state in real-time, so you can see exactly what is happening in your flow.

**Settings** — the icon in the top right corner. Click it to configure the node: which pin it uses, what values to compare against, how fast something repeats, and so on. Every node has its own settings.
