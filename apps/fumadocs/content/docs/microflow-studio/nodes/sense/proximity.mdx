title: Proximity
description: Read the distance to an object using various proximity sensor types.
---

A proximity sensor measures how far away an object is. It outputs a distance value that you can use to trigger actions based on how close something (or someone) gets.

**Example uses:**
- Trigger an action when someone's hand gets within 20 cm
- Display how far away an obstacle is
- Create an alarm that activates when something enters a zone

The node outputs a distance in centimetres.

## Types of proximity sensors
There are different types of proximity sensors available. The most common types are:

### Pulse/PWM
_No pulse/PWM proximity sensors have been tested with `Microflow` yet._

### Analog
_No analog proximity sensors have been tested with `Microflow` yet._

### Infrared

| Sensor                        | Range (Close) | Range (Long) | Resolution |
|-------------------------------|---------------|--------------|------------|
| GP2Y0A21YK                    | 0-10cm        | 10-80cm      | 1cm        |
| GP2D120XJ00F                  | 0-4cm         | 4-30cm       | 1cm        |
| GP2Y0A02YK0F                  | 0-15cm        | 15-150cm     | 1cm        |
| GP2Y0A41SK0F                  | 0-4cm         | 4-30cm       | 1cm        |
| GP2Y0A710K0F                  | 0-50cm        | 50-500cm     | 1cm        |

### I2C
_No I2C proximity sensors have been tested with `Microflow` yet._
