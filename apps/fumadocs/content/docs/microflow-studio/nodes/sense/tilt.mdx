---
title: Tilt
description: Detect the orientation of an object as either up or down using a tilt sensor.
---

A tilt sensor detects whether an object is tilted or upright. It gives you a simple yes/no answer: either the object is in its normal position, or it has been tilted past a certain angle.

**Example uses:**
- Detect when a box or container has been knocked over
- Trigger an alert when something should stay upright but gets tilted
- Use the orientation of an object as an on/off switch in your project

## Output values

The Tilt node outputs one of two states:
- **Up** — the sensor is in its normal, upright position
- **Down** (or tilted) — the sensor has been tilted

## Settings

Open the settings panel to configure:

- **Pin** — which digital pin on your Arduino the tilt sensor is connected to
