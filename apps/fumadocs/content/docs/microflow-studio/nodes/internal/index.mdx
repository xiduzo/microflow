---
title: Internal
description: Internal nodes help you understand and document your flow — monitoring and notes.
---

Internal nodes do not affect how your flow runs. They are tools for you as you build — to watch values in real-time, debug unexpected behavior, or leave notes explaining what a part of the flow does.

| Node | What it does |
| --- | --- |
| [Monitor](/docs/microflow-studio/nodes/internal/monitor) | Displays a value as it flows through, in real-time — useful for debugging |
| [Note](/docs/microflow-studio/nodes/internal/note) | A text note you can place anywhere on the canvas — for documentation |
