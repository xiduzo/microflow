---
title: Constant
description: Provide a fixed number that never changes to use throughout your flow.
---

A Constant node provides a fixed number that never changes. Think of it as a number you set once and it stays the same.

**Examples:**
- Set a constant value of 100 to use as a threshold for comparisons
- Use a constant value of 5 to multiply sensor readings by
- Provide a fixed brightness level (like 128) for an LED

This is useful when you need a specific number in your flow that does not come from a sensor or change over time. You can combine it with the [`Calculate`](/docs/microflow-studio/nodes/shape/calculate) node to do math — for example, multiply a sensor reading by your constant value.

## Settings

Open the settings panel to configure:

- **Value** — the fixed number this node will always output
