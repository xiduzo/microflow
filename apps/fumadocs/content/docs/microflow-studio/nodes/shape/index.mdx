---
title: Shape
description: Shape nodes transform and process data as it flows through your project.
---

Shape nodes sit in the middle of a flow and change the data passing through them. They do not read from hardware or control outputs — they just process numbers and signals.

Use them to:

- Do math on sensor readings
- Convert a value from one range to another (e.g. 0–1023 to 0–255)
- Smooth out noisy sensor data so it does not jump around

| Node | What it does |
| --- | --- |
| [Calculate](/docs/microflow-studio/nodes/shape/calculate) | Performs basic math — add, subtract, multiply, divide |
| [Map](/docs/microflow-studio/nodes/shape/map) | Converts a number from one range to another |
| [Smooth](/docs/microflow-studio/nodes/shape/smooth) | Averages recent values to stabilize a noisy signal |
