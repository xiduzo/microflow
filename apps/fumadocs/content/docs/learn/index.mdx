These tutorials are the fastest way to learn Microflow. Each one is a short, hands-on lesson with a clear outcome. Work through them in order — every lesson builds on the last.

You will need a computer, an Arduino-compatible board, and a USB cable. Some tutorials need extra parts; each lesson lists what you need at the top.

## How to use these tutorials

- **Type the steps yourself.** Reading is not the same as doing.
- **Stop and look.** When a step says "the LED turns on," verify it before moving on. If something does not match, the next step probably will not work either.
- **Do not worry about understanding everything yet.** Tutorials are about *experience*. The [Concepts](/docs/concepts) section explains the *why*.

When you finish all five, you have used every category of node in Microflow at least once.
