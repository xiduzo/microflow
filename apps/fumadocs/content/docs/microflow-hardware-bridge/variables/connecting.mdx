---
title: Connecting variables
description: How to set up variables in Figma so they can be linked to your hardware.
---

When you open the Microflow Hardware Bridge plugin in a Figma file, it automatically creates a variable collection called **MHB** (short for Microflow Hardware Bridge).

**Only variables inside the MHB collection are connected to your hardware.** Variables in other Figma collections are ignored by the plugin.

---

## How to set it up

1. Open your Figma file and run the **Microflow Hardware Bridge** plugin
2. The plugin creates the `MHB` collection automatically in your variable panel
3. Add variables to the `MHB` collection — or move existing ones into it
4. Those variables are now available to link with your hardware in Microflow Studio

Once you have variables in the MHB collection, check the [manipulating variables](/docs/microflow-hardware-bridge/variables/manipulating) page to learn how to read and update them from your hardware.
