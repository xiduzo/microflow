NOTE
For Blink.cpp.hex, the digital pin for the LED is set to D4. 
This is because there is no onboard LED on the IMUduino linked to pin 13. 
D4 was chosen because it is conveniently close to a GND pin, for ease of plugging an LED in to test.

Have fun!
