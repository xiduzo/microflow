# Microflow

Microflow is a visual flow-based programming tool for hardware prototyping and IoT development. It enables users to create interactive flows by connecting nodes that represent hardware components (buttons, LEDs, sensors, servos), data transformations, and control logic.

## Core Capabilities

- Visual node-based editor built on React Flow
- Real-time collaboration via Yjs with WebSocket sync
- Native desktop app via Tauri with Arduino/hardware flashing support
- MQTT integration for IoT connectivity
- Hardware abstraction through Firmata protocol

## Target Users

Makers, educators, and developers prototyping hardware projects without writing low-level code.
